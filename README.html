General

These notes assume that the Family Tree Project folder is copied and filed in total, including all associated folders, images and styles.

The Family Tree project is centred around Mary Elizabeth Kellett (nee Brown) 1927-2020

The reasoning for this is both personal and Mary, being an only child, gives a good pivot to the past and the future.


Structure

On each selected page there will be a similar structures with a header, content and links to all generations.

Any text highlighted in blue is a link.

When a link is hovered it will turn crimson and the cursor will change to a pointer

The links will take you to either a photo, gallery, document record or another person record.

Any element in a gallery can be selected and will display the document/photo selected.

On the each page, there is an email link. Please use this give me further information or any mistakes.
